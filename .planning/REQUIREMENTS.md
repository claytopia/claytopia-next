# Requirements: Claytopia

**Defined:** 2026-01-16
**Core Value:** Make visitors want to become part of the Claytopia community

## v1.0 Requirements (Homepage & Foundation)

### Homepage

- [ ] **HOME-01**: Hero section with "Time to get dirty" slogan and atmospheric image
- [ ] **HOME-02**: Overview of 4 main offerings (Clay Club, Brennservice, Workshops, Gruppen-Events)
- [ ] **HOME-03**: Teaser images and short descriptions for each offering (full sections with links)
- [ ] **HOME-04**: Clear CTA "Komm in den Club!" linking to contact
- [ ] **HOME-05**: Footer with contact info, address, and quick links

### Design & UX

- [ ] **UX-01**: Mobile-first responsive design
- [ ] **UX-02**: Warm, authentic visual design (earthy tones, handcrafted feel)
- [ ] **UX-03**: Clear, intuitive navigation (hamburger menu)
- [ ] **UX-04**: Consistent header/footer across all pages
- [ ] **UX-05**: Accessible design (proper contrast, semantic HTML)

### Technical

- [ ] **TECH-01**: Image optimization using Next.js Image component
- [ ] **TECH-02**: SEO meta tags on all pages
- [ ] **TECH-03**: Open Graph tags for social sharing
- [ ] **TECH-04**: Local business structured data (Schema.org)
- [ ] **TECH-05**: Lighthouse performance score > 90
- [ ] **TECH-06**: DSGVO-compliant cookie banner

## v1.1 Requirements (Content Pages)

### About

- **ABOUT-01**: Introduction to Pia Kadasch (owner/operator)
- **ABOUT-02**: Claytopia story and history
- **ABOUT-03**: Philosophy and approach to pottery
- **ABOUT-04**: Atelier photo gallery

### Clay Club

- **CLUB-01**: Detailed description of Clay Club concept
- **CLUB-02**: Opening times (Tuesday/Thursday evenings, Friday mornings)
- **CLUB-03**: Pricing and conditions
- **CLUB-04**: What's included (materials, tools, firing service)
- **CLUB-05**: FAQ section for common questions
- **CLUB-06**: Gallery of participant work
- **CLUB-07**: Contact option for signup/inquiries

### Brennservice

- **BRENN-01**: Service description explaining ceramic firing
- **BRENN-02**: Price list (by size/weight)
- **BRENN-03**: Process explanation (drop-off, pickup, timing)
- **BRENN-04**: Technical details (temperatures, kiln specifications)
- **BRENN-05**: FAQ section
- **BRENN-06**: Contact option for firing orders

### Workshops

- **WORK-01**: Overview of available workshops
- **WORK-02**: Workshop categories/themes
- **WORK-03**: Individual workshop details (description, dates, duration, price, capacity)
- **WORK-04**: Gallery of past workshops
- **WORK-05**: Contact option for booking

### Gruppen-Events

- **EVENT-01**: Description of group event options
- **EVENT-02**: Example scenarios (JGA, kids birthdays, team building)
- **EVENT-03**: Information about flexible arrangements and external locations
- **EVENT-04**: Testimonials/references from past events
- **EVENT-05**: Photo gallery of past events
- **EVENT-06**: Inquiry form for custom quotes

### Kontakt

- **CONTACT-01**: Contact form (name, email, message, subject/category)
- **CONTACT-02**: Phone number display with click-to-call
- **CONTACT-03**: Email link (hello@claytopia.de)
- **CONTACT-04**: WhatsApp link for quick contact
- **CONTACT-05**: Google Maps integration showing location
- **CONTACT-06**: Address and directions
- **CONTACT-07**: Opening hours/availability information

### Legal

- **LEGAL-01**: Impressum page with required legal information

## Out of Scope

| Feature | Reason |
|---------|--------|
| Online payment/booking system | Keep v1 simple with contact-based flow; Pia prefers personal communication |
| Newsletter system | Can add later, not critical for launch |
| User accounts/login | Not needed for contact-based model |
| Real-time chat | Contact form and WhatsApp sufficient |
| Video content | Photos sufficient for v1 |
| Multi-language | German audience only |
| Mobile app | Web-first approach |
